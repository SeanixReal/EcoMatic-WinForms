# 06 - Increment 1 Checklist and 3-Minute Video Guide

## Increment 1 Checklist

1. Project builds with no errors
2. Main menu opens correctly
3. Customer flow works (add item, insert money, purchase)
4. Admin flow works (login and restock)
5. Receipt can be shown after purchase
6. Documentation folder is complete
7. Mermaid diagrams are included

## Suggested 3-Minute Video Script

### 0:00 to 0:20 - Intro

1. Introduce project name and purpose
2. State that Increment 1 focuses on setup and UI design

### 0:20 to 1:30 - System Demo

1. Show MainForm
2. Open CustomerForm and add products
3. Insert money and complete purchase
4. Return to main and show receipt on exit flow

### 1:30 to 2:20 - Admin Demo

1. Open admin login
2. Enter password
3. Show inventory grid
4. Restock one item

### 2:20 to 3:00 - Documentation and Next Step

1. Show doc folder files
2. Show Mermaid flowchart and class diagram
3. Mention that MySQL integration is planned for Increment 2

## Tips for Recording

1. Keep windows zoom readable
2. Speak slowly and clearly
3. Focus on working features, not unfinished parts
4. Keep transitions smooth and short
